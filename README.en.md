# 简锐API开放平台

#### Description
基于React + Spring Boot + Dubbo + Gateway的API接口开放调用平台。管理员可以接入并发布接口，可视化各接口调用情况;用户可以开通接口调用权限、浏览接口及在线调试，并通过客户端SDK轻松调用接口。
